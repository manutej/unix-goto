## 🎯 What You Can Do

### Add Notes
```bash
goto-note "Fixed authentication bug in login.py"
```

### Search Notes
```bash
goto-recall "authentication"
# Finds your note instantly!
```

### View Notes for Folder
```bash
goto-what
# Shows all notes for current folder
```

### Get Folder Stats
```bash
goto-summary
# Shows visit count and recent notes
```
